# Shell Basics

## If its the same, then its the same. otherwise order is zsh/linux and then powershell

- Print a value to the console
```$zsh
echo 'Hello World'
```

- Create a file
```$zsh
touch filename
```

```$pwsh
New-Item [-Path] <path>
```

- Go down one folder level

```$zsh
cd folderToGoInto
```

- Go Up one folder level

```$zsh
cd ..
```
